Natural Language Toolkit

 nltk @

The NL Toolkit (NLTK) is a Python package intended to simplify the task of programming natural language processing systems. Its primary audience is graduate and undergraduate students studying computational linguistics.

Documentation

Several different types of documentation are available, describing different aspects of the toolkit.
  • Tutorials teach students how to use the toolkit, in the context of performing specific tasks. They are appropriate for anyone who wishes to learn how to use the toolkit.
  • The toolkit's reference documentation describes every module, interface, class, method, function, and variable in the toolkit. This documentation should be useful to both users and developers. Reference documentation is available for:
    • nltk 1.0 (released version.)
    • nltk 1.1a (development version.)
    • nltk_contrib (contributions package.)
  • A number of technical reports are available. These reports explain and justify the toolkit's design and implementation. They are used by the developers of the toolkit to guide and document the toolkit's construction. Students can consult these reports if they would like further information about how the toolkit is designed and why it is designed that way.

Download

Official releases of the toolkit are available from the SourceForge download page. The Getting Started guide gives detailed instructions for installing and using NLTK.

Open Source

NLTK is an open source project, currently released under the GNU Common Public License. All source code and documentation may be freely distributed and modified, in accordance with the license. NLTK is a member of OpenNLP, an umbrella project for open source NLP projects. See the OpenNLP webpage for more information, and to find other open source NLP projects.

Contribute to NLTK

There are several ways to contribute to NLTK.

  • Report bugs, Suggest features, or Contribute patches.
  • Participate in discussions on the NLTK-Devel Mailing List, or in the Public Forums.
  • Submit your NLTK-based projects for inclusion in the nltk_contrib directory, or provide a new module for inclusion in NLTK (must be relevant to NLP, substantial, original and well-documented). Please contact Edward Loper and Steven Bird.

People who have contributed sourcecode or documentation to NLTK: Chen-Fu Chiang, Jinyoung Choi, Trevor Cohn, Nikhil Dinesh, Brent Gray, Yurie Hara, Ewan Klein, Christopher Maloof, David Zhang.

Courses using NLTK

NLTK was designed for use in CIS-530 at the University of Pennsylvania, beginning in 2000. It has since been used in courses at several other institutions.

  • University of Pennsylvania: CIS-530 Introduction to Computational Linguistics
  • University of Melbourne: 431-460 Human Language Technology
  • University of Pittsburgh: CS1573 Artificial Intelligence Application Development
  • Simon Fraser University: CMPT 413 Computational Linguistics
  • Macquarie University: COMP348 Intelligent Text Processing
  • University of Edinburgh: Introduction to Computational Linguistics

Please tell us if you teach natural language processing using NLTK.
